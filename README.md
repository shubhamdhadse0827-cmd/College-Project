E-Building Care System
(Building Management System)

Project Overview :
E-Building Care System is a web-based Building Management System developed as a College Major Project under the academic curriculum.
The system provides a centralized platform to manage tenants, employees, payments, and building-related records efficiently.
This project aims to reduce manual work, improve data accuracy, and provide a user-friendly interface for building administration.

Objectives : 
-> Centralized management of building data
-> Efficient handling of tenants and employees
-> Secure and structured payment management
-> Easy profile and record maintenance
-> Practical implementation of Java Web Technologies

Key Features :
-> User Authentication & Role Management
-> Tenant Management
-> Employee Management
-> Payment & Billing Management
-> Profile & Record Management
-> Centralized Dashboard
-> Secure Database Connectivity
-> Responsive Web Interface

Technologies Used :
-> Java                  =	Backend Logic
-> JSP / Servlet         =	Server-side Processing
-> Maven                 =	Dependency Management
-> HTML, CSS, JavaScript = Frontend UI
-> MySQL                 = 	Database
-> Apache Tomcat         = 	Application Server
